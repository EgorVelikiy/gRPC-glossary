# gRPC сервис для управления глоссарием терминов

## Описание

Приложение представляет собой сервер, написанный на Python с использованием протокола gRPC. Сервис предоставляет API для управления глоссарием терминов, связанных с управлением состоянием в React-приложениях. Глоссарий состоит из терминов с описанием.

Присутствуют методы для реализации следующих действий:
* получение списка всех терминов,
* получение информации о конкретном термине по ключевому слову,
* добавление нового термина с описанием,
* обновление существующего термина,
* удаление термина из глоссария.

## Установка

### Шаг 1. Клонирование репозитория и переход в папку

### Шаг 2. Создание виртуального окружения и установка зависимостей

```bash
# создание виртуального окружения
python -m venv venv

# активация виртуального окружения (Windows)
venv\Scripts\activate

# установка зависимостей
pip install -r requirements.txt
```

### Шаг 4. Генерация proto-файлов

Перед запуском сервера необходимо сгенерировать Python-файлы из proto-определений:

```bash
python -m grpc_tools.protoc -I src/protobufs --python_out=src/glossary --grpc_python_out=src/glossary src/protobufs/glossary.proto
```

После выполнения этой команды в папке `src/glossary` будут созданы файлы:
- `glossary_pb2.py`
- `glossary_pb2_grpc.py`

## Запуск проекта

В двух разных терминалах необходимо активировать виртуальное окружение и затем запустить в одном сервер, в другом - клиента.

### Терминал 1 - Запуск сервера

```bash
# активация виртуального окружения
venv\Scripts\activate

# запуск сервера
python src/server.py
```

### Терминал 2 - Запуск клиента

```bash
# активация виртуального окружения
venv\Scripts\activate

# запуск клиента
python src/client.py
```
